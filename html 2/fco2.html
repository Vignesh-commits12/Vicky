1. HTML

    HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures the content on the web, allowing developers to define elements like headings, paragraphs, images, links, and more.

  2. CSS

    (Cascading Style Sheets) is a stylesheet language used to control the presentation and layout of HTML documents. It allows developers to apply styles to elements, enhancing the visual appearance of webpages.

  3. JS

    JavaScript (JS) is a versatile programming language primarily used for enhancing interactivity and dynamic content on webpages. It enables developers to create rich, engaging user experiences by allowing the manipulation of HTML and CSS in real-time.
